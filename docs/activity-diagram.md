# نمودار Activity - فرآیند عضویت در باشگاه

```mermaid
flowchart TD
    A([شروع]) --> B[کاربر وارد سایت می‌شود]
    B --> C[انتخاب گزینه 'عضویت جدید']
    C --> D[پر کردن فرم اطلاعات شخصی]
    D --> E{اعتبارسنجی اطلاعات}
    
    E -- ✔ اطلاعات صحیح --> F[نمایش تعرفه‌ها و شرایط]
    E -- ✘ اطلاعات ناقص --> D
    
    F --> G[انتخاب نوع عضویت<br>ماهانه/سه‌ماهه/سالانه]
    G --> H[هدایت به درگاه پرداخت]
    H --> I{پرداخت موفق؟}
    
    I -- ✔ بله --> J[ثبت عضویت در سیستم]
    I -- ✘ خیر --> K[نمایش خطای پرداخت]
    
    J --> L[ارسال تاییدیه ایمیل و پیامک]
    K --> M[بازگشت به صفحه پرداخت]
    M --> H
    
    L --> N([عضویت تکمیل شد])
    
    style A fill:#90EE90
    style N fill:#FFB6C1
    style E fill:#FFA500
    style I fill:#FFA500
    ```
